# Notes

To start the Flask server:

Linux & Mac:

```bash
export FLASK_APP=flaskr
export FLASK_NEW=development
flask run
```

Windows cmd:

```cmd
set FLASK_APP=flaskr
set FLASK_ENV=development
flask run
```

Windows Powershell:

```ps
$env:FLASK_APP = "flaskr"
$env:FLASK_ENV = "development"
flask run
```
